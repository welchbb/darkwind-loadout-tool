Extract a loadout

Upload one character's full settings export, then pick the aliases, highlights, triggers, timers, or functions you want to carry over. You'll get a small JSON file containing only what you selected.

Merge into a character

Upload the target character's full settings file, then one or more extracted loadout files to fold in. Anything that would collide with an existing alias, trigger, highlight, or timer gets flagged so you can decide which one wins.
